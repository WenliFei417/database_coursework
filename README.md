# Database Systems Coursework

This repository contains my coursework for a database systems class, including lab assignments and projects focused on SQL, database design, and backend data processing.

---

## 📁 Repository Structure

- **Labs_and_projects_instructions/**  
  Contains the original instructions and requirements for each lab and project.

- **Labs_submitted/**  
  Includes all submitted lab assignments.  
  These labs focus on fundamental database concepts such as:
  - SQL queries  
  - Joins and aggregations  
  - Stored procedures  
  - Database schema usage  

- **Projects_submitted/**  
  Contains three major projects completed during the course.  
  These projects involve more comprehensive tasks such as:
  - Database design and implementation  
  - Data validation and processing  
  - Backend logic using SQL and procedural code  
  - Handling real-world data scenarios
