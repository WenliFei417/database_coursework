## 🧠 Skills & Topics Covered

The labs are designed as a step-by-step progression, building from foundational SQL to advanced database programming:

### 🔹 Fundamental SQL & Querying
- Basic `SELECT` statements
- Filtering with `WHERE`, `ORDER BY`
- Working with different data types

### 🔹 Data Manipulation
- `INSERT`, `UPDATE`, `DELETE`
- Handling real data scenarios

### 🔹 Table Design & Joins
- Table creation and schema design
- Primary & foreign keys
- INNER JOIN, LEFT JOIN, multi-table joins

### 🔹 Views & Abstraction
- Creating reusable views
- Simplifying complex queries

### 🔹 Stored Procedures & Functions
- Parameterized stored procedures
- User-defined functions
- Business logic implementation in SQL

### 🔹 Advanced Database Programming
- Triggers for automatic data handling
- Cursors for row-by-row processing
- Error handling using TRY...CATCH
- Transaction management (COMMIT / ROLLBACK)
